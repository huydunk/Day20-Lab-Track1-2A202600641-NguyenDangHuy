## Recovery Path

Chọn **ít nhất một** trường hợp:

- [ ] User chưa có đủ dữ liệu
- [ ] User từ chối permission
- [ ] Kết quả đầu tiên chưa phù hợp
- [ ] Action thất bại hoặc bị gián đoạn
- [ ] User chưa sẵn sàng hoàn thành core action
